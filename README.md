# Gossamer

Gossamer is a modern e-commerce frontend built with Next.js, focusing on performance, responsiveness, and a clean user experience.

**Live Demo**: [https://gossamer-nu.vercel.app](https://gossamer-nu.vercel.app)

## Overview

This project is an actively developed online store client application featuring product catalog browsing, search functionality, filtering, and cart management.  
It leverages a contemporary tech stack to deliver a fast and scalable frontend solution.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Backend/Database**: Supabase
- **Deployment**: Vercel
- **Tools**: ESLint, Prettier

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/defosu-dev/gossamer.git
   cd gossamer
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env.local` file based on `.env.example` and fill in the required variables (Supabase URL, anon key, etc.).

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
